# Rules for adding tasks

**Adding tasks will be done in the [Tasks list](Task_list.md) file** \
Each task must contain \
Checkbox \
Task title \
List of action \
Image that explains the tas \
Assignee: the other student \
Status: starts as new 

**After a task enters working mode, it will enter [Tasks in progress](task_in_progress.md)** \
The title of the task is entered into the file \
Status change should be made in Task_list.md to in progress 

**[Results file](results.md)** \
The file is intended for actions that need to be performed in tasks
